PROGRAM
-------
timedatepair - Sample shows the user to understand the usage of ConvertTextToTIMEDATEPAIR
           & ConvertTIMEDATEPAIRToText.


ENVIRONMENTS
------------
Windows (32-bit)
Windows (64-bit)
Linux 64bit


FILES
-----
readme.txt   - This file specifies how to use this example.

timedatepair.c   - The main program.

mswin32.mak     - Make file for Windows 32-bit.

mswin64.mak     - Make file for Windows 64-bit.

linux64.mak     - Make file for Linux 64-bit.


RUNNING schedule:
-----------------

- Under Windows, type
  the following at the system command prompt:

  timedatepair 
  
- Under Linux, type the following at system command prompt:
  timedatepair

EXAMPLE - output
----------------------------------------
  timedatepair

  
After Text to TIMEDATEPAIR Lower: 03/09/2021 10:37:00 AM - Upper: 03/10/2021 10:37:00 AM
TIMEDATEPAIR to Text: 03/09/2021 10:37:00 AM - 03/10/2021 10:37:00 AM

Program completed successfully.
