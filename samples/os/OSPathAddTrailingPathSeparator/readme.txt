PROGRAM
-------
 OSPathAddTrailingPathSeparator - Adds separator at the end of path if the last character is not a path separator.

ENVIRONMENTS
------------
Windows (64-bit) 
Linux (64-bit) 

FILES
-----
OSPathAddTrailingPathSeparator.c    - Main program.

mswin64.mak     - Makefile for Windows 64-bit.

linux64.mak    - Makefile for Linux 64-bit.

readme.txt   - This file specifies the requirements for this example.


RUNNING  OSPathAddTrailingPathSeparator
-----------------------
- Under Windows or Linux,
  type:
       OSPathAddTrailingPathSeparator
  at the system command prompt.


The program should print output that looks like this:

 initializing Notes.

Domino Log Directory D:\IBM\Domino12\Domino\Data\IBM_TECHNICAL_SUPPORT\ Exists.

 Program completed successfully.
